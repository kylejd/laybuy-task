# Next.js + Jest

Example application with spoofed authentication.

## How to Use

### Run application

```bash
npm dev
# or
yarn dev
# or
pnpm dev
```

### Run Jest Tests

```bash
npm dev
# or
yarn test
# or
pnpm test
```

### Run e2e Playwright Tests

#### Setup Playwright

```bash
npx playwright install

npx playwright install-deps
```

```bash
npm test:e2e
# or
yarn test:e2e
# or
pnpm test:e2e
```
